Python Console

The python console lets you inerface with the python bindings for TrLib allowing you to automatise trivial tasks by simple python scripts.
You can extend the functionality by placing plugins (as usual python packages, see http://docs.python.org/2/tutorial/modules.html ) in a folder named:
~/.gsttrans/plugins 
On windows, the users home directory ~ is typically: C:\Documents and Settings\<my_user_name>
