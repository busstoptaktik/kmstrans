=== ONLY RELEVANT FOR PREPACKAGED VERSIONS ===

Packaged versions of this software for windows were built with py2exe: http://www.py2exe.org/
	
Unzip and place the unzipped folder where you'd like to keep it.

In order to run the program, you'll need also a valid geoid library. Download, unzip and point to the unzipped folder when the program
prompts for a geoid folder.
